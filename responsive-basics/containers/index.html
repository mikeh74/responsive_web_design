Container

The containers that we are referring to here are part of the Bootstrap CSS framework. They are used to wrap content and apply a max-width to it.

They are not an inherent part of the CSS specification but are commonly used in web development to keep content from stretching too much on large screens.

Bootstrap does provide a container type that will increase to fill the viewport width on larger screens, but generally this isn't what we want.

Containers stop line length getting too long and making it hard to read content. They also help to keep content centered on the page.

It also helps to ensure that relatively sized elements don't start getting too large on large screens. Items which are sized based on viewport units can get very large on large screens.

Container queries that are part of the CSS spec are very different the bootstrap concept of containers. They allow you to apply styles based on the size of the container rather than the viewport.
