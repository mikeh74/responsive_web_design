Responsive Display forced 100% width

If we make the width so that it's always 100% then the image will continue to grow past it's normal dimensions to fill whatever space is available.

Aspect Ratios

Aspect ratios in CSS are relatively new and only supported by ~95% of users browsers

  • 16 x 9

  • 3 x 2

  • 5 x 4

  • 1 x 1

Summary

  • By default images are not responsive in web browsers
  • The image should be larger than the space that it is intended to fill on the screen
  • Consider the likely maximum size of the display, it can be hard to provide standard guidance on this, this is the nature of responsive design
  • Aim for an aspect ratio that the image is inteded to be displayed at, but bear in mind that aspect ratio could change
  • Because images generatlly keep their aspect ratio fixed content displayed alongside them should be carefully considered
  • Media queries can be used to stop images from scaling up too large but again think about the maximum display
