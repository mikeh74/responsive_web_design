Basic CSS styling

Basic CSS styling is a good starting point for any project

Box A

This is a simple box with a border, some padding and a margin

      
        .box-a {
          border: 1px solid #000;
          padding: 1rem;
          margin: 1rem 0;
        }

Box B

This is a simple box with a border, some padding and a margin, also with a defined width and a background-color

      
        .box-b {
          border: 1px solid #000;
          padding: 1rem;
          margin: 1rem 0;
          width: 50%;
          background-color: #f8d7da;
        }

Example of position relative

Box C is inside this box

Box C

This is a simple box with a border, some padding and a margin, moved out from the normal flow

      
        .box-c-container {
          border: 1px solid #000;
          width: 50%;
        }
    
        .box-c {
          border: 1px solid #000;
          padding: 1rem;
          margin: 1rem;
          position: relative;
          top: -1rem;
          left: 3rem;
          width: 40%;
        }
