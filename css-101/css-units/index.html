Examples of CSS Units

Sizing with REMs

REM stands for "root em". It is a unit of measurement that is relative to the root element's font-size. The root element is the <html> element. REMs are useful for creating a consistent layout that scales well across different screen sizes.

By default, the root font-size is 16px. This means that 1rem is equal to 16px. If you change the root font-size to 20px, then 1rem will be equal to 20px.
